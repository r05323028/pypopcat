## Prerequisites

- Download [Chrome driver](https://chromedriver.chromium.org/) based on your OS.
- Move the driver to folders in `PATH` such as `/usr/local/bin`, `/usr/bin`.

## Installation

```bash
poetry install
```

## Usage

```bash
poetry run pypopcat emit --n_threads 1 --country TW
```
